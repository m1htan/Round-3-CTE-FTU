# Steps by Steps – Stock Alert System

## 1. Chuẩn bị môi trường
pip install -r requirements.txt

## 2. Step 1 – Thu thập dữ liệu EOD
Chạy lệnh để lấy dữ liệu EOD (OHLCV + fundamental cơ bản).
python main/step_1/fetch_eod.py
Kết quả: file CSV lưu trong data/step1_eod/

## 3. Step 2 – Sinh tín hiệu giao dịch
Tính toán chỉ báo kỹ thuật (TI), fundamental (FI), áp rule BUY/SELL.
python main/step_2/step_2.py --emit-latest-only
Kết quả:
- signals_eod_full_YYYYMMDD.csv
- alerts_eod_YYYYMMDD.csv
→ Lưu trong data/step2_signals/

## 4. Step 3 – Dashboard cảnh báo
Chạy dashboard Streamlit để theo dõi tín hiệu mới nhất.
streamlit run main/step_3/streamlit_app.py
Hiển thị danh sách cổ phiếu BUY/SELL theo ngày.

## 5. Step 4 – Backtest & Stress Testing
Đánh giá hiệu quả và kiểm thử hệ thống trong các kịch bản sốc thị trường.
# Backtest lịch sử
python main/step_4/backtest.py

# Shock test (mô phỏng tin tức bất ngờ, biến động lớn)
python main/step_4/shock_test.py

Báo cáo kết quả xuất tại data/reports/

Hoàn tất pipeline end-to-end.
